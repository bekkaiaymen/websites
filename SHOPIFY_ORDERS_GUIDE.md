# 📦 نظام إدارة طلبيات Shopify مع التأكيد والتصدير

## ✨ الميزات الجديدة

### 1️⃣ **عرض طلبيات Shopify الكاملة**
- كل طلبية تعرض **كاملة التفاصيل**:
  - اسم العميل
  - رقم الهاتف
  - الولاية والعنوان
  - المبلغ الكلي
  - المنتجات
  - رقم التتبع

### 2️⃣ **تأكيد الطلبيات**
- قبل الرفع إلى شركة التوصيل، **يجب تأكيد الطلبية** بعد التواصل مع العميل
- زر "✅ تأكيد" يظهر فقط للطلبيات غير المؤكدة
- الطلبيات المؤكدة تنتقل إلى حالة `pending` وجاهزة للتصدير

### 3️⃣ **تصدير إلى Excel**
- يتم تصدير **فقط الطلبيات المؤكدة**
- صيغة الـ Excel تتطابق تماماً مع متطلبات شركات التوصيل (Ecotrack):
  ```
  الرقم | الاسم | الهاتف | البلدية | الولاية | العنوان | اسم المنتج | المبلغ
  ```
- يمكن تحديد طلبيات معينة أو تصدير كل المؤكدة دفعة واحدة

### 4️⃣ **إحصائيات فورية**
- إجمالي الطلبيات
- عدد الطلبيات المؤكدة ✅
- الطلبيات بانتظار التأكيد ⏳
- الطلبيات المشحونة والمستلمة

## 📝 خطوات الاستخدام

### Step 1: الدخول لصفحة طلبيات Shopify
1. اذهب إلى `/admin/shopify-orders` أو `/merchant/shopify-orders`
2. ستشاهد جدول بجميع الطلبيات

### Step 2: تصفية الطلبيات
- **الكل**: عرض جميع الطلبيات
- **مؤكدة ✅**: فقط الطلبيات المؤكدة
- **بانتظار التأكيد ⏳**: الطلبيات التي تحتاج تأكيد

### Step 3: البحث
- ابحث عن العميل بالاسم
- ابحث برقم الهاتف
- ابحث برقم التتبع

### Step 4: تأكيد الطلبية
1. تواصل مع العميل وتأكد من التفاصيل
2. اضغط على زر **"✅ تأكيد"**
3. الطلبية تنتقل إلى حالة "مؤكدة"

### Step 5: تحديد للتصدير
1. الطلبيات المؤكدة تظهر مع **checkbox**
2. اختر الطلبيات التي تريد تصديرها
3. أو اضغط **"تحديد المؤكدة"** لتحديدها جميعاً

### Step 6: تصدير إلى Excel
1. اضغط على زر **"تصدير إلى Excel"**
2. سيتم تحميل ملف Excel يحتوي على:
   - جميع البيانات المطلوبة
   - بصيغة توافقية مع شركات التوصيل
3. الطلبيات المصدرة تنتقل إلى حالة `shipped`

## 🔧 معالجة المشاكل

### المشكلة: الطلبية لم تظهر
**الحل:**
1. تأكد من أن merchantId صحيح
2. تأكد من وجود token في localStorage
3. افحص console في DevTools للأخطاء

### المشكلة: زر التأكيد لا يعمل
**الحل:**
1. تأكد من وجود اتصال بالإنترنت
2. حدّث الصفحة (F5)
3. تحقق من أن الطلبية لم تكن مؤكدة بالفعل

### المشكلة: لا يمكن التصدير إلى Excel
**الحل:**
1. تأكد من وجود طلبيات مؤكدة
2. تأكد من أن رقم الهاتف بصيغة صحيحة
3. حاول تحديد طلبية واحدة أولاً

## 🚀 واجهات برمجة التطبيقات API

### الحصول على الطلبيات غير المؤكدة
```http
GET /api/orders/unconfirmed?merchantId={merchantId}
Authorization: Bearer {token}
```

### الحصول على الطلبيات المؤكدة
```http
GET /api/orders/confirmed?merchantId={merchantId}
Authorization: Bearer {token}
```

### تأكيد طلبية
```http
POST /api/orders/{orderId}/confirm
Authorization: Bearer {token}
Content-Type: application/json

{
  "adminId": "{adminId}"
}
```

### تصدير إلى Excel
```http
POST /api/orders/export-excel
Authorization: Bearer {token}
Content-Type: application/json

{
  "merchantId": "{merchantId}",
  "orderIds": ["{orderId1}", "{orderId2}"] // اختياري
}
```

### الحصول على الإحصائيات
```http
GET /api/orders/stats?merchantId={merchantId}
Authorization: Bearer {token}
```

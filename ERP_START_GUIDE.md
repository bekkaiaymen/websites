# 🚀 تشغيل نظام ERP بنجاح ✅

## 📍 الرابط الرئيسي:
```
👉 http://localhost:5173
```

---

## 🔐 بيانات تسجيل الدخول:

| الحقل | القيمة |
|------|--------|
| **اسم المستخدم** | `admin` |
| **كلمة المرور** | `admin123` |

---

## 📊 ما يمكنك الوصول إليه:

### 1️⃣ **لوحة التحكم الرئيسية** (Dashboard)
- 📈 إحصائيات عامة
- 💰 حالة المحفظة
- 📋 آخر الفواتير

### 2️⃣ **إدارة الطلبات** (Orders)
- 📦 قائمة الطلبات
- ✏️ تعديل الحالة
- 🗑️ حذف طلبات

### 3️⃣ **إدارة المنتجات** (Products)
- ➕ إضافة منتجات
- ✏️ تعديل المنتجات
- 📸 إدارة الصور

### 4️⃣ **الميزات الجديدة - نظام ERP** 🎯

#### 💰 المحفظة (Wallet)
- شراء الدولار بسعر 251 د.ج
- توزيع على التجار بسعر 330 د.ج
- متابعة الرصيد والأسعار

**الرابط**: `http://localhost:5173/admin/wallet`

#### 👥 إدارة التجار (Merchants)
- إضافة تجار جدد
- إعدادات مالية مخصصة
- حذف وتعديل

**الرابط**: `http://localhost:5173/admin/merchants`

#### 📄 الفواتير (Invoices)
- توليد فاتورة تلقائية
- تحميل الفاتورة Excel
- متابعة الحالة

**الرابط**: `http://localhost:5173/admin/invoices`

## ⚙️ الخصائص الرئيسية:

✨ **حسابات دقيقة** - خوارزمية FIFO  
✨ **تقارير شاملة** - فواتير Excel بالعربية  
✨ **أمان عالي** - JWT Authentication  
✨ **سهولة الاستخدام** - واجهة عربية  
✨ **مرونة** - تخصيص لكل تاجر

## 🐛 في حالة المشاكل:

**إذا لم تشاهد الصفحة**:
```bash
# تحقق من الخادم
curl http://localhost:5000/

# أعد التشغيل
npx kill-port 5000 5173
npm run dev
```

**إذا لم تتمكن من الدخول**:
```bash
# أنشئ مدير جديد
cd server
node init-admin.js
```

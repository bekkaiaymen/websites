# مشروع توصيل غرداية - نسخة احترافية

مشروع توصيل احترافي وكامل مع نظام المصادقة والـ PWA ليتم تحميله على الهاتف.

## ✨ المميزات الرئيسية

- ✅ **نظام تسجيل دخول وحساب شخصي** - كل عميل يملك حسابه الخاص
- ✅ **لوحة تحكم شخصية (Dashboard)** - لإنشاء وتتبع الطلبات
- ✅ **تطبيق PWA** - يمكن تحميله على الهاتف مثل التطبيقات العادية
- ✅ **قاعدة بيانات قوية** - تخزين آمن لكل المعلومات
- ✅ **واجهة احترافية** - سهلة الاستخدام والملاحة

## 🚀 البدء السريع

### 1. تشغيل محلياً (على جهازك)

```bash
# الدخول إلى مجلد المشروع
cd e:\delivery

# تثبيت المكتبات
npm install

# تشغيل الخادم والواجهة معاً
npm run dev
```

ثم افتح الرابط: `http://localhost:5173`

### 2. البيانات المطلوبة

قبل الرفع إلى الاستضافة، تأكد من أن لديك:

- ✅ حساب MongoDB Atlas (قاعدة البيانات) - مجاني
- ✅ حساب GitHub (لحفظ الكود)
- ✅ حساب Render (للاستضافة المجانية)
